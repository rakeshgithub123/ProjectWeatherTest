Steps to run the test -
	1. Download the repository from github from below - 
		https://github.com/rakeshgithub123/ProjectWeatherTest
	2. Extract the files to C:/ProjectWeatherTest
	3. Run the console application from C:/ProjectWeatherTest from command prompt using the below command -
		dotnet run NDTVWeather.exe
	4. Test results are printed on the console
	
Steps to set the variance-
	1. Use the config.json file to specify the variance for different parameters. The file located here after extraction to C:/ drive
		C:\ProjectWeatherTest\config.json
	
Things to note -
	1. .net core may have to be installed if not already installed in the system where the test will be executed
	2. Currently the project supports only chrome browser
